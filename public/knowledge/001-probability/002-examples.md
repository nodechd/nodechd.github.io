# Math Examples

This doc shows how to write inline and display equations in the knowledge base.

Inline math example: Euler's identity — $e^{i\pi} + 1 = 0$.

Display math example (quadratic formula):

$$
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$

Probability example (Bayes' theorem):

$$
P(A\mid B) = \frac{P(B\mid A)\,P(A)}{P(B)}
$$

Markdown source examples:

```markdown

$$

$$
```
